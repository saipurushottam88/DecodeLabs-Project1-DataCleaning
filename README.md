# DecodeLabs Project 1 - Data Cleaning & Preparation

## Internship Project
This project was completed as part of the DecodeLabs Data Analytics Internship.

## Objective
Clean and prepare a raw dataset by:
- Handling missing values
- Verifying duplicate records
- Standardizing date formats
- Checking data consistency
- Validating price calculations

## Tools Used
- Microsoft Excel

## Tasks Performed
- Filled blank CouponCode entries with "No Coupon"
- Verified duplicate records
- Standardized date formatting
- Validated TotalPrice calculations
- Improved dataset readability
